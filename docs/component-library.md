# Web UI 组件库文档

## 设计令牌

- 颜色
  - 主色：`#4f46e5`
  - 文本：`#0f172a`
  - 次级文本：`#475569`
  - 边框：`#e2e8f0`
- 圆角
  - 容器：`16px`
  - 轻量控件：`10px`
- 阴影
  - 主卡片：`0 8px 24px rgba(15,23,42,.05)`

## 基础样式类

- `surface-card`
  - 用于主卡片与容器块，统一边框、阴影、圆角
- `glass-banner`
  - 用于页面首屏横幅，表达品牌层次
- `rise`
  - 鼠标悬停抬升反馈
- `focus-ring`
  - 键盘焦点可视化
- `fade-in`
  - 首次进入区域过渡
- `muted`
  - 次级文案颜色

## 页面级组件

- Header Banner
  - 标题：Fanbox Extractor
  - 副标题：高性能下载管理与文件预览控制台
- 下载面板
  - 平台选择
  - 凭据输入
  - 创作者选择
  - 高级选项
  - 启停控制
  - 进度与日志
- 管理面板
  - 文件树工具区
  - 文件预览区

## 交互规范

- 主操作按钮使用高对比主色
- 危险操作使用负向色并保持独立位置
- 所有输入控件支持键盘可达
- 状态更新优先通过文本与进度条双通道反馈

## 响应式规范

- `<1024px`：操作区与内容区上下排列
- `>=1024px`：管理页左右分栏
- 移动端按钮默认满宽，桌面端恢复自适应宽度
